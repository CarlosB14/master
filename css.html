Componentes CSS
  • Selector--> Elemento o elementos que seleccionamos para darle estilo ej: selector { propiedad:valor; propiedad:valor; etc...} ej código: p { color:red;}

  • Valor--> Definimos el valor al que cambiamos

Incluir CSS
  • Arcivo CSS externo--> El código se escribe en un archivo .css aparte. La mejor forma.
  • Bloque de estilos--> El código se escribe en una etiqueta style en documento html
  • Estilos de línea--> El código se escribe en un atributo html style en una etiqueta

Selectores
  • Selector universal --> *--> Selecciona todos los elementos de la página
  • Selector de etiqueta--> Selecciona todos los elementos de la misma etiqueta
  • Selector clase--> Lo definimos como atributo a una etiqueta de elemento para diferenciarlo del resto y poder asignarle diferentes valores en CSS se define como .destacado {}, .destacado.grande{} puedes poner que cuando tenga esas dos clases añadirle algo más
  • Selector ID--> Identifica únicamente a un elemento, se define en CSS #destacado{}
  • Selector mixto--> Podemos combinar las selecciones --> p.destacado {} me sacará los párrafos que tengan la clase destacado
  • Seleccionar atributos básicos--> etiqueta[atributo] --> Sacará todas las etiquetas que tengan ese atributo
  • Selector de atributos avanzados--> Se puede complicar todo lo que queramos, úclase selectores 1 en los últmos min salen los que existen

  • Selector de hijo--> Descendiente de sólo 1 nivel, #page > div, para hacer sólo hijos directos, muy útil para no aplicarlo en todo.
  • Selector de hermanos--> Del mismo padre, por ejemplo: --> li ~ li, elegirá los li que tengan un hermano li, al primero no le aplicará el efecto sólo a los siguientes.
  • Selector de hermano adyacente--> Etiqueta hermana pero sólo a continuación del elemento especificado, 'a + b'

  • Pseudoclases de hijos--> strong:first-child, strong:last-child, elige primer hijo o último hijo. li:nth-child(3) elige el tercer li. li:nth-last-child(2) elige el penúltimo li:nth-child(2n) cada dos elementos, si es (3) elegirá cada 3. Modelos de cajas
  • Definición--> Todo en CSS está en una caja invisible, y lo podemos modificar. Borde, margen, relleno y contenido.
  • with--> Establecer ancho de un elemento, sólo funciona con elementos de bloque. auto o ?px
  • max-with, min-with--> El máximo y mínimo que puede ocupar un elemento, mas reestrictivo que with
  • height--> El alto de un elemento, sólo con elementos de bloque
  • max-height, min-height--> El máximo y mínimo que puede ocupar el elemento, más reestrictivo que height
  • margin--> Espacio entre dos cajas, left, right, top, bottom, se puede simplificar margin: 25px 50px 75px 40px Según cuantos metamos se establece de una forma. Si chocan 2 márgenes de diferentes cajas, coge la de mayor px.
  • padding--> Igual que en margin. Para establecer relleno en la caja, nunca se ocuparía.
  • border--> border-width, border-style, border-color, tmb se puede poner con top, left etc, tiene varios estilos. Se puede definir los 3 a la vez, border: 5px dotted purple; border-radius: 25px, para darle la esquina redondeada
  • box-sizing--> Le das el tamaño con todo ya aplicado a la caja entera, con padding y border. box-sizing: border-box
  • opacity--> va del 0 al 1, por defecto es 1, 0 es invisible
  • overflow--> Para el texto que se desborda de la caja. Se puede definir en general o en x e y, visible, hidden, scroll, auto

Posicionamiento--> Súper importante
  • display--> Para cambiar un elemento de bloque a línea o viceversa, atributo, block, inline e inline-block, elemento de línea pero que se le puede poner width y height, se puede usar none para que no se vea
  • vertical-align--> Para posicionar un elemento de línea dentro de un párrafo por ejemplo arriba, abajo o middle, por ejemplo para subíndices o ªº
  • position, relativo es igual que normal pero con desplazamiento, absoluto cambias origen del elemento, fijo siempre está en la misma posición, aún haciendo scroll etc, flotante ya no se usa, sticky para cuando haces scroll fijar el menú por ejemplo
    • normal--> normal
    • relativo--> es igual que normal pero con desplazamiento --> position: relative; left: 100px; si pones -100 es como poner 100 en right
    • absoluto--> cambias origen del elemento, SUPER POTENTE, --> su eje de coor, es el primer ancestro que está posicionado de forma no estático, las demás cajas ocupan su lugar al contratio que el relativo
    • fijo--> siempre está en la misma posición, aún haciendo scroll etc, toma de referencia la ventana del navegador, y no se mueve aún haciendo scroll
    • sticky--> sticky para cuando haces scroll fijar el menú por ejemplo, para que funcione tiene que haber scroll en la etiqueta padre
    • z-index--> Para decir qué elemento queremos que esté por encima, con posicionamiento diferente a por defecto, por ejemplo absoluto, relativo fijo..., si lo aplicamos tiene que ser sobre mismo tipo de posicionamiento
