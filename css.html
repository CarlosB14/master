Componentes CSS
  • Selector--> Elemento o elementos que seleccionamos para darle estilo ej: selector { propiedad:valor; propiedad:valor; etc...} ej código: p { color:red;}
  • Propiedad--> Qué es lo que quiero modificar
  • Valor--> Definimos el valor al que cambiamos
  • Comentario--> /**/ todo lo que esté dentro de los asteriskos es comentario
  • Regla--> Par de propiedad valor
Incluir CSS
  • Arcivo CSS externo--> El código se escribe en un archivo .css aparte. La mejor forma.
  • Bloque de estilos--> El código se escribe en una etiqueta style en documento html
  • Estilos de línea--> El código se escribe en un atributo html style en una etiqueta
Unidades de medida
    Unidades absolutas--> Principal px, poco flexibles pero fácil de usar sin cálculos intermedios. Y usar pt para impresión.
    • with--> Ancho
    • font-size-->
    Unidades relativas
    • %--> Su tamaño relativo a su elemento padre
    • em--> Relativo al font-size de su elemento padre, se multiplica por los px de su elemento padre
    • rem--> Relativo al elemento font-size principal
      • viewport--> Relativo al tamaño del navegador
      • vw--> Ancho del viewport 50vw es 50% del navegador
      • vh--> Alto del viewport
      • vmin--> Para hacer elementos cuadrados
      • vmax--> Para hacer elementos cuadrados
    Colores CSS
    • color--> Cambia color de texto que está en el interior de un elemento
    • background-color--> Cambia el color de fondo de un elemento
    • Lista colores-- redFalta url y eliges el nombre del color
    • RGB--> (0,0,0) rojo azul y verde

Tipos de etiquetas y CSS del navegador
  • Etiquetas de bloque--> Van a ocupar el máximo que tenga de ancho el navegador
  • Etiquetas de línea--> Sólo ocupan su contenido
  • Normalize.css--> Para dar un valor a los estilos CSS de nuestra web igual en todos los dispositivos
Hojas de estilo en cascada
  • Cascada--> Algoritmo o reglas que cumple el navegador para aplicar CSS
  • Orden aparición--> 1--> Atributo style en etiqueta HTML. 2--> bloque de estilos en documento HTML. 3 --> archivo .css externo
  • Herencia--> Si se establece en un elemento padre todos los hijos heredan ese estilo.
Selectores
  • Selector universal --> *--> Selecciona todos los elementos de la página
  • Selector de etiqueta--> Selecciona todos los elementos de la misma etiqueta
  • Selector clase--> Lo definimos como atributo a una etiqueta de elemento para diferenciarlo del resto y poder asignarle diferentes valores en CSS se define como .destacado {}, .destacado.grande{} puedes poner que cuando tenga esas dos clases añadirle algo más
  • Selector ID--> Identifica únicamente a un elemento, se define en CSS #destacado{}
  • Selector mixto--> Podemos combinar las selecciones --> p.destacado {} me sacará los párrafos que tengan la clase destacado
  • Seleccionar atributos básicos--> etiqueta[atributo] --> Sacará todas las etiquetas que tengan ese atributo
  • Selector de atributos avanzados--> Se puede complicar todo lo que queramos, úclase selectores 1 en los últmos min salen los que existen
  • Selector descendente--> Seleccionar elementos unos dentro de otro, con cuidado, a b, separamos el elemanto a del b con un espacio. 'div#page div' --> Seleccionamos los div que sean descendientes de, los div que tengan id page
  • Selector de hijo--> Descendiente de sólo 1 nivel, #page > div, para hacer sólo hijos directos, muy útil para no aplicarlo en todo.
  • Selector de hermanos--> Del mismo padre, por ejemplo: --> li ~ li, elegirá los li que tengan un hermano li, al primero no le aplicará el efecto sólo a los siguientes.
  • Selector de hermano adyacente--> Etiqueta hermana pero sólo a continuación del elemento especificado, 'a + b'
  • Selector múltiple--> Se separa con , podemos aplicarlo a todo lo que queramos, al poner la , es como decir y añadiendo el siguiente
  • Pseudoclases de hijos--> strong:first-child, strong:last-child, elige primer hijo o último hijo. li:nth-child(3) elige el tercer li. li:nth-last-child(2) elige el penúltimo li:nth-child(2n) cada dos elementos, si es (3) elegirá cada 3. (2n + 1) Coge los impares. (odd) coge impares, (even) coge los pares
  • Pseudoclases de hijos según tipo elemento--> :first-of-type, .container div:first-of-type --> primer elemento div de la clase container aunque no sea el primer hijo. Se aplica todo lo demás con child
  • :only-child, :only-of-type--> Cuando son hijos únicos. *:empty cualquier elemento que no tenga hijos o contenido. Por ejemplo una imagen
    • Pseudoclases de interacción
    • :hover--> Cuando pasas el ratón por encima
    • :active--> Cuando cualquier usuario pincha el elemento, pero se activa justo cuando está pinchando
    • :focus--> El elemento que hemos pinchado por última vez
    • :focus-within--> Si uno de sus elementos hijos tiene el focus
    • :focus-visible--> Cuando tiene el focus de forma visible
  • Pseudoselectores de ubicación--> :link --> Enlaces sin visitar :visited--> enlaces visitados
