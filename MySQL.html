Comandos
  • SHOW DATABASES--> Nos da las bases de datos en este servidor
  • Crear base datos--> CREATE database nombre-bbdd;
  • USE bbdd--> Nos situamos en la base de datos
  • SHOW TABLES--> Muestra las tablas de la base de datos que nos hemos situado
  • SELECT database();--> Nos devuelve la bbdd que estamos ahora mismo
  • SELECT COLUMNA/S FROM TABLA;--> Para seleccionar columna en una tabla y una database ya situados en ella
  • limit 50--> Muestra 50 filas, pero procesa todas
  • Operar con el dato de una columna--> SELECT Name, Population*2 from city;
  • Asignamos un alias en la respuesta--> SELECT Name, Population*2 AS "Double Population" from city;
  • DISTINCT--> SELECT DISTINCT CountryCode FROM city; --> Saca todos los cod de país sin repetir
  • Asegurar bbdd para cambios--> select... FROM world.city --> Así entramos seguro a la bbdd que podemos modificar
  • Where--> SELECT * FROM world.country WHERE Continent="Asia";
  • Comparador <> --> WHERE Name > 'za' --> sólo saca los nombres que siguen a za...
  • WHERE BINARY--> Tiene que estar escrito exactamente igual
